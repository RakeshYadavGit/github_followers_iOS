## Overview

**GitHub Followers iOS** is an open-source iOS application that allows users to search GitHub usernames and view their followers. The app is built using Swift and follows modern iOS development practices.

## Features

- Search GitHub users by username.
- View a list of followers for any GitHub user.
- See follower details such as avatar, bio, and profile link.
- Clean and intuitive UI.
- Handles network requests and error gracefully.
